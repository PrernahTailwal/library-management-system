# Library Management System (SQL)

This project is a MySQL-based Library Management System designed to manage
books, authors, categories, publishers, and issue/return transactions.

## Features
- Database design using primary and foreign keys
- Table relationships
- Issue and return management
- Fine calculation for overdue books
- SQL queries using JOIN, GROUP BY, and aggregate functions

## Technologies Used
- MySQL
- SQL

## Files
- PROJECT.sql : Contains database creation, tables, sample data, and queries

## Usage
1. Open MySQL Workbench
2. Run the PROJECT.sql file
3. Execute queries as required
